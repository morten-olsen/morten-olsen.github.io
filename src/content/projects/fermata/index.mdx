Fermata is a media player built to restore intentional listening. It consolidates music, podcasts, and audiobooks into a single ad-free interface backed by self-hosted services — no algorithms, no cloud accounts, no tracking.

## Features

- **Music** — browse Jellyfin libraries with prominent album artwork, playlist creation, and gapless playback
- **Podcasts & audiobooks** — connect via Audiobookshelf with bidirectional progress sync and chapter navigation
- **Cross-device playback** — route audio to speakers via Home Assistant integration
- **Offline-first** — local library syncing so everything works without a connection
- **Cross-platform** — runs on iOS, Android, and web
